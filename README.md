# About
My Debian boilerplate image that forms the base for my docker containers.

Dockerfile to build a Debian:8(`sid-[mirrors ustc]`) baseimage with a couple of extra packages.

 Extra packages && Main packages:

- `python3`
- `python3-apt`
- `python3-pip`
- `python3-dev`
- `redis-server`
- `libffi-dev`
